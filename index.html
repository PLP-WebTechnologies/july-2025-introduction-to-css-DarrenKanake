What is CSS?

CSS stands for Cascading Style Sheets. It is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS controls the layout of multiple web pages all at once.

Why Use CSS?

  • Separation of content and style
  • Improved website performance
  • Consistency across pages
  • Ease of maintenance

Basic Syntax

CSS is made up of selectors and declarations. A selector points to the HTML element you want to style. A declaration consists of a property and a value.

Basic Styling Properties in CSS

Here are some common CSS properties you can use to style your web pages:

  • color: Sets the text color.
  • background-color: Sets the background color of an element.
  • font-size: Sets the size of the font.
  • margin: Sets the space outside an element.
  • padding: Sets the space inside an element.
  • border: Sets the border around an element.
  • width: Sets the width of an element.
  • height: Sets the height of an element.
  • text-align: Sets the horizontal alignment of text.
  • display: Specifies how an element is displayed (e.g., block, inline, flex).

CSS Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree and consists of margins, borders, padding, and the actual content.

  • Content: The actual content of the box, where text and images appear.
  • Padding: Clears an area around the content; inside the border.
  • Border: A border that goes around the padding (if any) and content.
  • Margin: Clears an area outside the border; it is transparent.
